### LVGL

This is the LVGL example from the [Random Nerd Tutorials site](https://randomnerdtutorials.com/lvgl-cheap-yellow-display-esp32-2432s028r/) web site. All the intructions for setting it up are on that web site.

This is the way LVGL is usually used on the ESP32. It uses the TFT_eSPI library to provide the display driver.
